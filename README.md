# WCCI 2026 Special Session (Simple GitHub Pages)

This is a minimal, Google-Sites-like single-page website for the WCCI 2026 special session:
**Secure, Fair, and Privacy-Preserving Optimization and Learning**.

## Deploy on GitHub Pages
1. Create a GitHub repository.
2. Upload all files from this folder to the repository root.
3. GitHub → Settings → Pages → Deploy from a branch → `main` / `(root)`.
4. Your site will appear at: `https://<username>.github.io/<repo>/`

Tip: keep `.nojekyll` so GitHub Pages won’t run Jekyll.
